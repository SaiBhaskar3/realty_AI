# UrbanIQ – US Neighborhood Insights

UrbanIQ is a Streamlit-based data science application that compares two US locations
across:

- Real Estate Features  
- Safety Index  
- Quality of Life Metrics  
- Semantic Search of Federal Building Dataset  

### Features
✓ Clean UI  
✓ Radar charts, bar charts, building data  
✓ Optional API integrations (Census, WAQI, Zillow, FBI Crime API)  
✓ Embedding-based semantic search
